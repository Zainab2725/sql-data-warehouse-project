## ğŸ—ï¸ Data Architecture
The data architecture for this project follows the **Medallion Architecture** with **Bronze, Silver, and Gold layers**:

- **Bronze Layer:** Stores raw data as-is from source systems. Data is ingested from CSV, JSON, or other structured files into SQL Server.  
- **Silver Layer:** Data cleansing, normalization, and standardization are performed here to prepare data for analytics.  
- **Gold Layer:** Contains business-ready data, modeled into a **star schema**, optimized for reporting and decision-making.
